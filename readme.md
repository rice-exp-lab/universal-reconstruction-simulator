# Synthetic Multi-View Dataset Generation for 3D Reconstruction

This project proposes a Unity-based platform for generating synthetic multi-view datasets and evaluating 3D reconstruction methods such as **3D Gaussian Splatting**.

The platform allows the creation of configurable camera rigs, controlled acquisition environments, and synthetic datasets for computer vision research.

This work is part of a capstone project developed at **Rice University**.

---

# Project Overview

The goal of this project is to develop a simulation framework that allows researchers to:

- Generate synchronized multi-view datasets
- Simulate realistic camera configurations
- Evaluate 3D reconstruction algorithms
- Validate camera calibration pipelines in a controlled environment

The system is implemented in **Unity HDRP** and integrates **OpenCV for Unity** for camera calibration.
