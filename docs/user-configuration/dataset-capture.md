# Dataset Capture

**Object:** Capture  
**Script:** `SceneCapture`

## Goal
Capture an image dataset from all cameras.

## Execution
In Play Mode, activate the capture option in the component.

## Parameters
- **Camera Rig** — Parent containing cameras
- **Width / Height** — Output resolution
- **Capture FPS** — Frames per second
- **Num Frames** — Total frames to capture
- **Run Name** — Output folder name, saved inside `Project/Capture`

## Example
If **FPS = 2** and **Num Frames = 10**, the total capture duration is **5 seconds**.
